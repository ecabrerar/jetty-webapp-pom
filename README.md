jetty-webapp-pom
================

Minimal Maven 3 pom.xml for webapps using jetty

#### Getting the project


```bash
git clone https://github.com/ecabrerar/jetty-webapp-pom.git
```

To start quickly, build the project :
```bash
cd jetty-webapp-pom
mvn clean install
```
and start the web application with jetty :

```bash
mvn jetty:run
```
